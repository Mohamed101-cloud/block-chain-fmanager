# Blockchain File Manager

## 📌 Description

This project is a file management system integrated with blockchain technology to ensure security and integrity of data.

## ⚙️ Features

* Upload and manage files
* Blockchain-based verification
* Local database storage
* Simple user interface

## 🛠️ Technologies Used

* Python(in python i ues tkinter for ui)
* Blockchain (Solidity / Ganache)
* SQLite
* VS Code


## 📂 Project Structure
*login_ui.py (هذا ملف تسجيل الدخول)
* `File_Manager_ui.py` → (هذا واجهة رفع الملفات)
* `file.py` → (ملف الباكند لرفع الملفات)
* `Blockchain_back.py` → (ملف البلوكتشين لا تدخله اذا مو اختصاصك)
* `file_database.py` → database
